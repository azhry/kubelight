## Development Setup

### Prerequisites

- Node.js 18+
- Docker (for integration testing)
- A frontend package manager (npm / yarn / pnpm)
- Backend runtime (Node.js / Python / Go / etc.)

### Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Run tests: `npm test`

### Running with the Harness

This project is managed by the `agent-spec-ops` harness. To start
a new delivery session:

```bash
cd agent-spec-ops
node scripts/read-context.js runs/KUBELIGHT-001/workflow-state.json
```

## Architecture

- **Frontend**: Located in the frontend directory
- **Backend**: Located in the backend directory
- **Harness**: `agent-spec-ops/` contains the delivery management system
